변수 선언 
 
 var : 함수영역 
 선언된 함수내애서 사용 가능 
 함수가 종료 될때까지 사용 가능 
 
 
 let : 블럭영역 
 선언된 블럭이 종료되면 변수의 사용도 종료된다 
 
 for(let i= 0 ; i<=10; i++){
	 nums += i  + "  ";
 }
  let i변수는 for문 안에서만 사용 가능하다
 
 
 const : 변하지 않는 값을 지정
 const pi = 3.141592;
 pi의 값은 처리도중 값을 변경할 수 없다 


지역변수  :변수가  선언된 함수내에서만  사용 가능 
var out ;

전역변수  : 변수가 선언된 함수 뿐만 아니라 다른 함수에서도  사용가능 
out
